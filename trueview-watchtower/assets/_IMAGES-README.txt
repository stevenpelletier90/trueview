IMAGES THIS THEME EXPECTS
=========================

The four image files below were NOT bundled automatically: the Claude Design
connector caps file downloads at 256 KiB and all four of these exceed it, so
pulling them would have produced corrupt files. They are your own brand assets
(you uploaded them to the design project), so just copy them into this folder
with these exact filenames.

Drop these into  wp-content/themes/trueview-watchtower/assets/  :

  logo.png    True View Security Solutions logo.
              Used in the header (~30px tall) and footer (~24px tall).
              In the design this is your "TrueView" logo on a transparent
              or white background (the header/footer sit it on white).

  unit.png    Cut-out photo of the mobile surveillance unit / trailer.
              Home page → "The Unit / One trailer. Total coverage." section.
              Displayed up to 420px wide; a transparent PNG looks best.
              (Design source: the trailer "unit" image.)

  tw-24.jpg   Home hero photo — a unit deployed on site.
              Right-hand image in the hero, cropped to fill (object-fit:cover).
              (Design source: TrueView_Watchtower-24.jpg)

  tw-28.jpg   Home "Wide-Area Coverage" full-width band photo.
              Full-bleed background behind the navy overlay.
              (Design source: TrueView_Watchtower-28.jpg)

Until these are in place, the Home page will show broken-image icons where the
photos go. The Contact page needs only logo.png.

Tip: large JPEGs/PNGs slow the page down — run them through an optimizer
(e.g. squoosh.app) and aim for well under ~300 KB each before uploading.
